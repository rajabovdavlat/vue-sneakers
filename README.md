# Vue 3 Sneakers Shop

A fully functional online sneakers store built with Vue 3, TypeScript, Tailwind CSS and Vite.

**Live Demo**: [https://vue-sneakers-mauve.vercel.app](https://vue-sneakers-mauve.vercel.app)

## Features
- Dynamic product catalog loaded from mock API (axios)
- Sorting by name and price
- Real-time search
- Add to favorites and cart with visual feedback
- Sliding cart drawer with total price and 5% tax
- Fully responsive design (mobile-first)

## Tech Stack
- Vue 3 (Composition API + `<script setup>`)
- TypeScript (full typing)
- Vite
- Tailwind CSS
- Axios
